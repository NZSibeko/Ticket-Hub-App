## 📋 Key Files for Deployment

### Must Configure Before Deployment

| File | Purpose | Action Required |
|------|---------|-----------------|
| `backend/.env` | Environment variables | Copy from `.env.production.example`, fill in secrets |
| `nginx/nginx.conf` | Reverse proxy | Update domain name |
| `backend/ecosystem.config.js` | PM2 config | Update deploy section with your repo |
| `scripts/deploy-huawei.sh` | Deployment script | Review and customize if needed |

### Generated Files (Don't Edit)

| File | Generated By | Notes |
|------|--------------|-------|
| `backend/dist/` | TypeScript build | Created by `npm run build` |
| `backend/node_modules/` | npm install | Created by `npm install` |
| `backend/prisma/schema.prisma` | Prisma | Edit schema, then run `prisma generate` |

### Git-Ignored Files (Never Commit)

```
.env
.env.production
backend/logs/*
nginx/ssl/*
backups/*
node_modules/
dist/
.expo/
*.log
```

## 🎯 Deployment Flow

```
1. Development (Local)
   └── npm run dev
   └── Test locally

2. Staging (Optional)
   └── Deploy to staging ECS
   └── Test with production-like setup

3. Production (Huawei Cloud)
   ├── Provision infrastructure (ECS, RDS, OBS)
   ├── Run deploy script or manual deployment
   ├── Configure environment variables
   ├── Start application with PM2
   ├── Setup Nginx + SSL
   └── Update mobile app API endpoint
```

## 📊 Technology Stack

### Frontend (Mobile)
- **Framework**: React Native 0.81.4
- **SDK**: Expo ~54.0.12
- **Navigation**: React Navigation 7.x
- **State**: React Context + Hooks
- **UI**: Custom components + React Native SVG
- **Payments**: Stripe React Native
- **Camera**: Expo Camera / Vision Camera
- **i18n**: React i18next

### Backend (API)
- **Runtime**: Node.js 20.x
- **Framework**: Express 5.x
- **Language**: TypeScript 5.9+
- **Database**: PostgreSQL (via Prisma ORM)
- **WebSocket**: Socket.IO 4.x
- **Auth**: JWT (jsonwebtoken)
- **Security**: bcryptjs, CORS, Helmet
- **Process Manager**: PM2

### Infrastructure (Huawei Cloud)
- **Compute**: ECS (Elastic Cloud Server)
- **Database**: RDS for PostgreSQL
- **Storage**: OBS (Object Storage)
- **Network**: VPC, EIP, ELB (optional)
- **Monitoring**: Cloud Eye
- **SSL**: Let's Encrypt via Certbot

## 🆘 Troubleshooting

### Common Issues

**Backend won't start:**
```bash
pm2 logs tickethub-backend --lines 50
# Check .env configuration
# Verify database connection
```

**Database connection failed:**
```bash
# Test from ECS
psql -h <rds-endpoint> -U tickethub_user -d tickethub
# Check security group allows ECS IP
```

**WebSocket not connecting:**
```bash
# Check Nginx config for /socket.io location
# Verify proxy_set_header Upgrade
# Test: curl -i http://localhost:8081/socket.io/
```

**High memory usage:**
```bash
# Check PM2 memory
pm2 monit
# Increase instance max_memory_restart
# Consider scaling ECS
```
